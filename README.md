## Running locally

* Install Ruby and Bundler.
  On Debian/Ubuntu : `apt-get install ruby bundler zlib1g-dev build-essentials`
* Run `bundle install`
* Run `bundle exec jekyll serve` to run the site locally. 
* Open browser on `http://localhost:4000`
